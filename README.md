# 绵阳产业通

## 安装项目依赖
```
yarn install
```

### 运行热启动开发环境
```
yarn serve
```

### 为生产编译和压缩
```
yarn build
```

### 打包并自动部署
```
yarn build-upload
需要配置服务器信息
```

### Lints and fixes files
```
yarn lint
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).

###
注意:
  在部署生产环境时，需要确认并修改.env.production DOMAIN和INDEX_URL两个变量
